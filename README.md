# AppleIGB
### Intel onboard LAN driver for macOS. Courtesy of [hnak](https://www.insanelymac.com/forum/profile/228503-hnak/?wr=eyJhcHAiOiJmb3J1bXMiLCJtb2R1bGUiOiJmb3J1bXMtY29tbWVudCIsImlkXzEiOjI3MzA3MywiaWRfMiI6MTc3NTc1Mn0=), refer to the [original post](https://www.insanelymac.com/forum/topic/273073-appleigbkext/) for more details. <br />Updated to Intel IGB source v5.2.7
<hr />

## Supported Devices

 - 82575
 - 82576
 - 82580
 - dh89xxcc
 - i210/i211
 - i350
 - i354

Updated version only tested on AMD versions of Intel I211-AT so far. 

No support given but feel free to fork and pull-request.
